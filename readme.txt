== Description ==

A motley collection of rare, fringe, bleeding-edge tweaks and workarounds that certain users of the  <a href="http://www.prophotoblogs.com/">ProPhoto</a> theme might want because of rare server issues, or because they're bat sh*t crazy.

== Installation ==

Upload the plugin to your blog, or install it through the Plugins page, then activate it. The plugin will add a new option page called "Tweaks Plugin" to the "ProPhoto" top-level option menu.  On that page you can manipulate various advanced tweaks and workarounds by filling out the form and saving changes.
